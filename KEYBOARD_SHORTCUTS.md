# ChatSphere Keyboard Shortcuts

Keyboard shortcuts for efficient navigation and interaction.

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Tab` | Move focus to next interactive element |
| `Shift + Tab` | Move focus to previous interactive element |
| `Enter` | Activate focused element |
| `Escape` | Close modal/dialog, cancel action |
| `/` | Focus search (when not in input) |

---

## Customizing Shortcuts

Currently, keyboard shortcuts are not customizable. This is a planned feature.

To request new shortcuts, open an issue on GitHub.

---

## Screen Reader Support

ChatSphere includes ARIA live regions for:
- New messages (announced automatically)
- Typing indicators
- Connection status changes
- Error messages

All interactive elements have proper ARIA labels and roles.

---

## Tips

1. **Focus management**: Use `Tab` to navigate efficiently
2. **Quick actions**: Learn the single-key shortcuts for frequent actions
3. **Accessibility**: All features are keyboard accessible
4. **Screen readers**: Live regions announce dynamic content
